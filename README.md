# 图书管理系统简介 java1034

## 项目概述

图书管理系统是一个基于鸿蒙操作系统（HarmonyOS）的前后端项目，主要实现了图书管理的增删改查功能，以及用户登陆和注册机制。

## 技术栈

- **前端**：使用 ArkTs 语言开发
- **后端**：基于 SpringBoot 框架，数据存储采用 MySQL

## 功能模块

- 用户登陆与注册
- 图书的增加、删除、修改和查询

## 系统角色

- 系统管理员：负责图书管理和用户管理
- 普通用户：可以进行图书查询和个人信息管理

## 运行环境

- HarmonyOS 版本：Api 9

## 部署步骤

1. 准备运行环境，确保 MySQL 数据库和 SpringBoot 服务器正常配置。
2. 部署后端服务，启动 SpringBoot 服务。
3. 部署前端应用，确保与后端服务正常通信。

## 目录结构

```
- frontend/          # 前端代码目录
  - assets/
  - components/
  - pages/
  - ...
- backend/           # 后端代码目录
  - src/
  - pom.xml
  - ...
```

## 核心亮点

- **独立开发**：本项目由单一开发者独立完成，展现了全面的技术能力。
- **数据持久化**：项目采用 MySQL 数据库保存数据，保证了数据的一致性和安全性。
